# ActFEMtide

## 概要

ActFEMtide は、火山構造イメージング用の3次元有限要素法(FEM)電磁探査コード [ActFEM](https://github.com/) をベースに、**潮汐起因の電磁場をソースとして扱えるように拡張したバージョン**です。

ActFEM はもともと以下の解析を対象としていました。

- 人工電流源(Active source)による電磁探査(振幅・位相応答)
- MT(Magnetotelluric)法による地下比抵抗構造探査
- Active source + MT + Tipper の結合インバージョン

ActFEMtide では、これらに加えて **海洋潮汐・地球潮汐に起因する起電力(潮汐発電効果)をソース項として組み込み**、潮汐周期で観測される電磁場変動をFEMで順解析・解釈できるようにすることを目的としています。

## 背景・目的

沿岸・海底の電磁観測点では、潮汐周期(半日周期・日周期など)を持つ電磁場変動がしばしば観測されます。これは海水の潮汐流が地球磁場中を運動することによる起電力(海洋潮汐発電効果)が主な要因と考えられますが、火山地域などでは地下比抵抗構造の影響も受けるため、定量的な解釈にはFEMによる順解析が有効です。

ActFEMtide は、このような潮汐起源の電磁場を

1. 海洋潮汐モデル(例: OTPS など)から潮汐流・水位変化を計算し、
2. これを起電力・電流源としてFEMメッシュに組み込み、
3. 任意の3次元比抵抗構造に対する電磁応答を計算する

というワークフローを実現することを目指しています。

## ActFEM との違い

| 項目 | ActFEM | ActFEMtide |
|---|---|---|
| 主なソース | 人工電流源 / 平面波(MT) | 潮汐起因の起電力 |
| 主な対象 | 能動ソース電磁探査、MT探査 | 潮汐周期電磁場応答 |
| 適用例 | 阿蘇火山などの構造イメージング | 沿岸・海底火山(例: 鬼界カルデラ)における潮汐応答解析 |
